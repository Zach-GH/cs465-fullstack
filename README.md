# cs465-fullstack
CS-465 Full Stack Development with MEAN

1. How to run: 
    - express --view=hbs --git -f
    - DEBUG=cs465-fullstack:* npm start

2. If you forget to check out to a new branch:
    - git stash (your changes)
    - git checkout -b *
    - git stash pop
